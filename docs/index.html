Converter imagem → matriz de bordado

Envie uma imagem em alta qualidade. Imagens pequenas podem perder detalhes e gerar costura ruim.

Atenção: a imagem parece ter resolução baixa. Use uma imagem maior/mais nítida.

Passo 1: Configuração base

Dica: prefira PNG/JPG com boa nitidez.
Mesma proporção (escala automática).
E3D não é suportado nesta versão.
Mais cores = mais trocas de linha.
Alto = mais pontos/arquivo maior.
Controla contorno, compensação e equilíbrio de densidade.
Local: http://127.0.0.1:8000 — Hospedado: URL do seu servidor.

Passo 2: Ajustar objetos

Edite objeto por objeto: cor, tipo de preenchimento, densidade, compensação e sob-costura.
